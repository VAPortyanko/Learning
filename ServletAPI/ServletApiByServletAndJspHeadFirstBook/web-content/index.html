Security

Basic authentication with Tomcat. Add next roles and users in the tomcatdir/conf/tomcat-users.xml:

<role rolename="Admin" />
<role rolename="Member" />
<role rolename="Guest" />
<user username="admin" password="admin" roles="Admin" />
<user username="member" password="member" roles="Member" />
<user username="guest" password="guest" roles="Guest" />

The page below is available only for a user with the "Admin" role:
